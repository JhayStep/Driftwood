# Driftwood — Offline‑First CRDT Sync Engine (Rust)

A production‑style offline‑first data sync engine built on CRDTs, with a reusable library (`crdt-core`), a networked sync node (`sync-node`), and a demo CLI (`demo-cli`).

## Build
```bash
cargo build
```
